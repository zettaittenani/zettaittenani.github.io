What I did

  • Took an English lesson in the morning.
    • Understood the difference between “I feel ~” and “I feel like ~”
    • We should use “I feel like ~” literally when I feel “like” (not directly but feel we feel next to some emotions)
    • So, when we’d like to express some virtual situations, we should use “I feel like ~”
      • For instance, “I felt like I was on cloud 9”.
    • When we’d like to express some direct emotions, we should use “I feel ~”
      • For example, “I felt so happy”.
